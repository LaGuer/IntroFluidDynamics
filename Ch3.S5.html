3.5 The Velocity Potential and Laplace’s Equation

For any irrotational flow, where

∇×𝐮=0,

a scalar function ϕ can be defined such that

𝐮=∇ϕ. (3.18)

Then, since the curl of a gradient is zero, irrotationality is automatically satisfied. Here, ϕ is called the velocity potential.

Alternatively, we could write the potential as

ϕ=∫𝒪𝒫𝐮⋅𝑑𝐱, (3.19)

where 𝒪 is an arbitrary point in the fluid. There’s a subtle point here, though, so be careful. As long as the region is simply connected – that is, the fluid has no holes in it – the path between 𝒪 and 𝒫 doesn’t matter, and ϕ will be a single-valued function.

However, if the region is multiply connected – it has holes, regions where there is no fluid – the integral could depend on the path and ϕ will be a multivalued function of position. Let’s do some examples to investigate this in detail.

  Example 3.2 – Uniform flow.

Let’s start with uniform flow, which is simple and one we’ll use extensively. Suppose 𝐮=(U,0,0), where U is the constant velocity of the flow in the x-direction. Using either equation (3.18) or (3.19), we can see that

ϕ=U⁢x=U⁢r⁢cos⁡θ. (3.20)

I’ve written the potential down in both Cartesian coordinates and also cylindrical; we’ll need the cylindrical version later on.  

  Example 3.3 – Flow past a stagnation point.

Recall the flow past a stagnation point, given by

𝐮=(α⁢x,-α⁢y,0).

This is irrotational flow (check if you want!), and to find the velocity potential, write

∂⁡ϕ∂⁡x=u=α⁢x and ∂⁡ϕ∂⁡y=v=-α⁢y.

Integrating and combining suggests

ϕ⁢(x,y)=12⁢α⁢(x2-y2)+c,

where c is the integration constant. Note, however, that the fluid velocity, as opposed to the potential, is the physically meaningful quantity; since we take a derivative to get to the velocity, we can set the constant c to zero without worry that it’s meaningful.

Note that here ϕ is a single-valued function of x and y. That means that, for any point in the fluid, ϕ has a single value.  

  Example 3.4 – Line vortex flow.

For our last example, consider the flow

𝐮=kr⁢θ^.

We have to be careful for this flow – it’s irrotational (we showed that back in Problem 1), but not at the origin where it blows up. To fix this problem, we’ll suppose there’s no fluid there – maybe there’s a cylinder of radius a there instead, covering up the problem area. That means the fluid domain is r≥a, but it also means there’s now a hole in the fluid domain; it’s multiply connected.

The potential is found from equation (3.18) in cylindrical coordinates,

∂⁡ϕ∂⁡r=ur=0,1r⁢∂⁡ϕ∂⁡θ=uθ=kr,and ∂⁡ϕ∂⁡z=uz=0.

Integrating this gives

ϕ⁢(θ)=k⁢θ. (3.21)

But note that this isn’t a single-valued function – ϕ has different values at the same point in space: ϕ⁢(0)=0, but ϕ⁢(2⁢π)=2⁢π⁢k, and so on. We’ll see later on that this is connected to circulation within the fluid.

 

One final thing before we do one more example. If the fluid is incompressible, it must satisfy the incompressibility condition,

∇⋅𝐮=0.

If we rewrite this in terms of the potential, we get

∇⋅∇⁡𝐮=0,

or

∇2ϕ=0. (3.22)

This is Laplace’s equation; any irrotational, incompressible fluid must satisfy it.
